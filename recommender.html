🎯 TubeKit AI 서비스란?

TubeKit AI는 세계 최고의 AI 기술인 Google Gemini, OpenAI ChatGPT, Anthropic Claude를 활용하여 사용자에게 최적의 유튜브 채널을 추천하는 무료 웹 서비스입니다.

기존의 유튜브 알고리즘은 시청 기록에만 의존하지만, 저희 서비스는 연령대, 성별, 지역, 콘텐츠 카테고리, 스타일, 영상 길이 등 다양한 요소를 종합적으로 분석하여 진정으로 당신에게 맞는 채널을 찾아드립니다.

3개의 서로 다른 AI 엔진이 동시에 분석하기 때문에, 한 가지 관점이 아닌 다양한 시각의 추천을 받을 수 있습니다. 게임, 음악, 교육, 요리, 여행, IT/기술, 재테크 등 14개 이상의 카테고리와 7가지 콘텐츠 스타일을 지원하며, 쇼츠부터 긴 영상까지 선호하는 영상 길이도 선택할 수 있습니다.

개인정보 보호를 최우선으로 생각하여, 입력하신 정보는 서버에 저장되지 않으며 추천 완료 후 즉시 삭제됩니다. 무료로 제공되는 이 서비스를 통해 새로운 유튜브 채널을 발견하고, 더 풍요로운 콘텐츠 생활을 시작해보세요!

❓ 자주 묻는 질문 (FAQ)

네, 완전히 무료입니다. 무료 사용 시 하루 3회까지 Gemini AI를 통한 추천을 받을 수 있습니다. ChatGPT나 Claude를 사용하시려면 본인의 API 키를 입력하시면 무제한으로 이용 가능합니다.

세 AI는 각각 다른 회사(Google, OpenAI, Anthropic)에서 개발한 대규모 언어 모델입니다. 각 AI는 학습 데이터와 알고리즘이 달라 같은 질문에도 다른 관점의 추천을 제공합니다. 여러 AI의 추천을 비교해보시면 더 다양한 채널을 발견할 수 있습니다.

입력하신 연령, 성별, 관심사 등의 정보는 AI 추천을 생성하는 데만 사용되며, 서버에 저장되지 않습니다. 추천이 완료되면 모든 데이터는 즉시 삭제됩니다. API 키를 입력하셨다면 브라우저의 로컬 스토리지에만 저장되어 서버로 전송되지 않습니다.

유튜브 알고리즘은 시청 기록 기반으로 추천하지만, 저희 서비스는 사용자가 직접 입력한 취향과 선호도를 기반으로 추천합니다. 새로운 분야의 채널을 찾거나, 기존 알고리즘의 필터 버블에서 벗어나고 싶을 때 유용합니다.

입력 정보를 더 구체적으로 수정하시거나, '추가 관심사' 필드에 더 자세한 키워드를 입력해보세요. 또한 다른 AI 엔진으로 다시 추천받으시면 다른 관점의 결과를 얻을 수 있습니다.

네, 반응형 웹으로 제작되어 스마트폰, 태블릿, PC 등 모든 기기에서 이용 가능합니다. 별도 앱 설치 없이 웹 브라우저로 접속하시면 됩니다.

Gemini API 키는 Google AI Studio(aistudio.google.com)에서, ChatGPT API 키는 OpenAI 플랫폼(platform.openai.com)에서, Claude API 키는 Anthropic Console(console.anthropic.com)에서 발급받을 수 있습니다. API 키 설정 버튼을 클릭하시면 각 사이트 링크도 제공됩니다.

현재 한국어, 영어, 일본어, 중국어 4개 언어를 지원합니다. 페이지 우측 상단의 언어 선택 버튼으로 변경할 수 있습니다. 추천 결과도 선택한 언어에 맞게 제공됩니다.
